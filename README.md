# TDF
my_tdf_calc
Dedicated python program to find total dose fraction(TDF) for radiotherapy treatment regime.
this program only guides the user to find the number of fractions left in the treatment regime .
tdf is the polished successor of NSD which is nominal standard dose.
the purpose of the making this program is get rid of tedious calculations on paper during the routine work in medical physics.
